# Main flow (Tab):

Home | Control | Automation | Profile

# Screen List

## Auth

Screen Login
Screen SignUp

## Tab Home

Screen Home: quick access to top 3 automation, usage time chart

## Tab control

Screen Control: Device list,Toggle ON/OFF
Screen Device Detail: fan, air conditioner, smart light, smart door

## Tab automation

Screen Automation: List automation, Consumption of the month, Device power consumption
Screen Detail of each automation
Screen Add new automation

## Tab Profile

Screen User Profile: user information, Usage time chart, Logout.
Screen Edit profile
Screen Change password
